                          # Intro

                          The theme provide comment support using vuepress-plugin-comment2, and it pass plugin.comment in theme options to the plugin.

                          vuepress-plugin-comment2 currently supports 3 comment services:

                          • Giscus
                          • Twikoo
                          • Waline
