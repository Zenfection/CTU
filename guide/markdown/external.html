                                                  # Markmap

                                                  Markmap is a tool for converting Markdown into mind maps. It supports more format and content than the built-in flowchart of the theme.

                                                  1. Use Markmapopen in new window to generate mind map HTML file
                                                  2. Place the HTML file under .vuepress/public/
                                                  3. Insert into Markdown via <iframe>

                                                  Input:

                                                  <iframe
  :src="$withBase('/markmap/demo.html')"
  width="100%"
  height="400"
  frameborder="0"
  scrolling="No"
  leftmargin="0"
  topmargin="0"
/>
